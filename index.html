API Basejump: Image Search Abstraction Layer

User story:
  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=20 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example usage:

https://twhou-image-search.herokuapp.com/api/search/cats
https://twhou-image-search.herokuapp.com/api/latest

About outputs:

  • Search will return 10 results.
  • If you want the next page, use ?offset=10 after the search term.
